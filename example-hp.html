Welcome to the Design system Boilerplate for Angular 15. It contains the following :

  • Angular 15
  • Design system Angular Component library version 15 (note: we do not use Standalone components...yet )
  • Tailwind 3.2 with Design system Tailwind configuration
  • root css styles with css vars for Arval and partners
  • Visual Assets: Arval, partner ( ex: autosphere )s and brands ( ex: Bmw ) logos in multi file formats ( svg, png and webp ) and flavors ( dark, flat, normal, mobile )
  • Font Assets : Bnpp-sans, bnpp-sans-condensed and open-sans ( with preconfigured css )
  • favicons : Arval and partners in multi format ( ico, Android, apple ) and multi sizes
  • Examples of components use ( header, footer, toasts, snack bar, tooltip, card, button )
  • Main layout ready to use ( this page )
